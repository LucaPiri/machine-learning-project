#Attenzione a non fare conflitto con i dati su github, prima di fare ogni modifica al file fare 
git pull
#in questo modo siete nella repo che sta su github

Quando dovete caricare dei file fate questo:
git add . #vi permette di aggiungere tutti i file e/o cambiamenti al vostro branch
git commit -a -m "inserire messaggio da far vedere agli altri" #con -a committate tutto nel vostro branch, con -m dite che ci mettete anche un messaggio 
git push #con questo mandate tutte le informazioni sul git
